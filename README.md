# 📊 Customer & Sales Analysis Dashboard

This project analyzes a retail dataset to uncover insights related to sales performance, customer behavior, and profitability using MySQL and Power BI.

---

## 🔧 Tools Used
- MySQL (data cleaning and transformation)
- Power BI (dashboard and visualization)
- DAX (calculated measures)

---

## 📊 Dashboard Overview

### 🔹 Page 1 — Sales Overview
- KPIs: Total Sales, Total Profit, Profit Margin, Number of Orders  
- Monthly sales trend  
- Category and regional performance  

### 🔹 Page 2 — Customer & Product Analysis
- Sales by customer segment  
- Top 10 products by sales  
- Sub-category performance  
- Ship mode filtering  

### 🔹 Page 3 — Profitability Analysis
- Scatter plot: Discount vs Profit relationship  
- Profit margin by sub-category  

---

## 🔍 Key Insights

- Higher discount rates are associated with lower profitability  
- Some high-revenue products generate low or negative profit  
- West region contributes the highest profit, while Central underperforms  
- Profit margins vary significantly across sub-categories
